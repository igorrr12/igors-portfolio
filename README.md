# Portfolio — Igor Matysiak

Prosta, statyczna strona portfolio (HTML / CSS / JavaScript). Bez frameworków i bez kroku budowania.

## Uruchomienie lokalne
Otwórz `index.html` w przeglądarce, albo uruchom prosty serwer:

```bash
python -m http.server 8000
# następnie otwórz http://localhost:8000
```

## Struktura
- `index.html` — treść strony (sekcje: O mnie, Projekty, Kontakt)
- `styles.css` — style i motyw
- `script.js` — animacje pojawiania się sekcji + rok w stopce

## Wdrożenie (GitHub Pages)
1. Wypchnij pliki do repozytorium `igorrr12/igors-portfolio`.
2. W ustawieniach repo: **Settings → Pages → Branch: `main` / root**.
3. Strona pojawi się pod adresem `https://igorrr12.github.io/igors-portfolio/`.
